# Төслийн товч танилцуулга — URL Shortener

## 1. Төслийн нэр

URL Shortener — REST API + Minimal Frontend

## 2. Төслийн ерөнхий тайлбар

Энэ төсөл нь урт URL холбоосыг богино холбоос болгон хувиргах жижиг систем юм. Хэрэглэгч урт URL оруулахад систем санамсаргүй богино код үүсгэж, тухайн кодыг эх URL-тай холбон хадгална.

Дараа нь хэрэглэгч богино холбоосоор хандахад систем тухайн кодыг шалгаж, эх URL руу redirect хийнэ.

Мөн систем нь богино холбоос дээр хэдэн удаа дарсныг тоолох click counter-тэй байна. Үүнээс гадна богино холбоос бүрт хугацаа дуусах огноо буюу expiration date тохируулж болно. Хэрэв холбоосын хугацаа дууссан бол систем redirect хийхгүй, алдааны мэдээлэл буцаана.

Энэхүү төсөл нь F.CSM311 Программ хангамжийн бүтээлт хичээлийн “Бие даалт 13 — AI-Assisted Software Construction” ажлын хүрээнд хийгдэнэ.

## 3. Төслийн гол зорилго

Энэ төслийн гол зорилго нь том хэмжээний production систем хийх биш, харин AI-тай хамтран программ хангамж боловсруулах бодит workflow-г турших юм.

Үүнд дараах алхмууд багтана:

- Spec буюу шаардлага тодорхойлох
- Generate буюу AI-аар санаа, код, бичиг баримт үүсгүүлэх
- Review буюу AI-ийн гаргасан зүйлийг шалгах
- Integrate буюу өөрийн төсөлд нэгтгэх
- Test буюу тестээр баталгаажуулах
- Reflect буюу хийсэн ажлаа эргэцүүлж тайлагнах

AI хэрэгслийг төлөвлөлт, кодын санаа, review, тестийн санаа, documentation болон reflection хэсгүүдэд ашиглана. Гэхдээ AI-аар үүсгэсэн бүх зүйлийг гараар шалгаж, ойлгож, засварлаж нэгтгэнэ.

## 4. Сонгосон сэдэв

Сонгосон сэдэв:

URL shortener — REST API + minimal frontend. Random short code, click counter, expiration.

## 5. Хамрах хүрээ

Энэ төсөл дараах үндсэн feature-үүдтэй байна.

### Feature 1 — Богино URL үүсгэх

Хэрэглэгч урт URL оруулна. Backend санамсаргүй богино код үүсгээд тухайн URL-тай холбон хадгална.

Жишээ:

```text
Урт URL: https://example.com/very/long/link
Богино код: aB12xY
Feature 2 — Богино кодоор redirect хийх

Хэрэглэгч дараах хэлбэрийн холбоосоор хандахад:

/{short_code}

систем тухайн богино кодод харгалзах эх URL руу redirect хийнэ.

Feature 3 — Click counter

Богино холбоос амжилттай нээгдэх бүрт тухайн холбоосын click count 1-ээр нэмэгдэнэ.

Feature 4 — Expiration

Богино холбоос бүр хугацаа дуусах огноотой байж болно. Хэрэв одоогийн огноо expiration date-аас хэтэрсэн бол систем redirect хийхгүй.

Feature 5 — Minimal frontend

Хэрэглэгчид зориулсан энгийн web page байна. Түүн дээр:

урт URL оруулах
expiration date оруулах эсвэл хоосон үлдээх
Shorten товч дарах
үүссэн богино URL-г харах

боломжтой байна.
